# 📚 BookStore

A modern, responsive online bookstore built using **HTML**, **CSS**, and **JavaScript**. This project was created to practice front-end web development fundamentals while focusing on clean design, responsiveness, and an intuitive user experience.

🌐 **Live Demo:**
https://yashaswini-1006.github.io/bookstore/

---

## ✨ Features

* 📖 Responsive landing page
* 🌙 Light/Dark mode toggle
* 📚 Book catalog layout
* 🎨 Modern and clean user interface
* 📱 Mobile-friendly design
* ⚡ Built with vanilla HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## 📂 Project Structure

```text
bookstore/
├── index.html

├── assets/

```

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yashaswini-1006/bookstore.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional dependencies or installation are required.
